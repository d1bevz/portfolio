# portfolio

Research-ассистент личного инвестиционного портфеля на базе Claude Code.

- **Брокер:** Revolut Invest (Metal) · **Резидентство:** Португалия · **Взнос:** ~100 €/мес (DCA)
- **Роль Claude:** глубокий рисёч + рыночные данные (Alpha Vantage MCP) + ежемесячные рекомендации.
- **Исполнение:** вручную в приложении Revolut. Все решения и ответственность — на инвесторе.

Правила работы ассистента — в [`CLAUDE.md`](./CLAUDE.md).

## Структура

| Файл / папка | Назначение |
|---|---|
| `CLAUDE.md` | Правила и гардрейлы для Claude (анти-галлюцинация, границы) |
| `positions.md` | Текущие позиции портфеля |
| `watchlist.md` | Кандидаты на покупку + триггеры |
| `theses/` | Инвестиционные тезисы (1 файл = 1 идея) |
| `decisions-log.md` | Лог всех решений с обоснованием |
| `reports/` | Ежемесячные DCA-отчёты |

## Источник данных

[Alpha Vantage MCP](https://mcp.alphavantage.co/) — OHLCV (день/неделя/месяц, 20+ лет) + фундаментал.

Подключение (нужен бесплатный ключ с https://www.alphavantage.co/support/#api-key):

```
claude mcp add -t http alphavantage "https://mcp.alphavantage.co/mcp?apikey=ВАШ_КЛЮЧ"
```

⚠️ Free-tier лимит ~25 запросов/день — достаточно для ежемесячного процесса по watchlist.

## Ежемесячный процесс

1. Запустить Claude в этом репо: «сделай ежемесячный DCA-отчёт».
2. Claude тянет live-данные по позициям и watchlist, сверяет тезисы, предлагает 2–4 покупки на 100 €.
3. Инвестор исполняет в Revolut → вносит факт в `positions.md` и `decisions-log.md`.

> Не является финансовой или налоговой консультацией.
